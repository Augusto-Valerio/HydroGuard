Tecnologias utilizadas

HTML

Linguagem de marcação usada para
estruturar o conteúdo da página,
como títulos, parágrafos, imagens e
links.

CSS

Linguagem de estilo usada para definir cores,
espaçamento, fontes e o layout visual da interface.

JavaScript

interatividade da página.

Mapas Integrados via API

Exibe rotas simuladas com base em dados de localização.

Arduino

É uma plataforma que possibilita o desenvolvimento de projetos eletrônicos.

WAI-ARIA

WAI-ARIA torna conteúdo e aplicações web mais acessíveis, auxiliando com conteúdo dinâmico e controles avançados em HTML e JavaScript.

Python

Responsável pela lógica e interatividade da página.
